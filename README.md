# Second topple bilibili
1. **2.1文件夹为2.1相关代码**[^1]
   1. src为代码相关文件夹
        1. 根目录下为html
        2. js文件夹内为js
        3. css文件夹下为css
   2. res为资源相关文件夹，photo文件夹下存放html中的图片
   3. 使用方法：
        1. 打开2.1videoplayer.html
        2. 点击右侧的“选择文件”按钮，选择想要播放的本地视频
        3. 选择后点击右侧边栏的“视频1”即可开始播放

# Second topple bilibili
1. **2.1文件夹为2.1相关代码**[^1]
   1. src为代码相关文件夹
        1. 根目录下为html
        2. js文件夹内为js
        3. css文件夹下为css
   2. res为资源相关文件夹
        1. photo文件夹下存放html中的图片
   3. 使用方法：
        1. 打开2.1videoplayer.html
        2. 点击右侧的“选择文件”按钮，选择想要播放的本地视频
        3. 选择后点击右侧边栏的“视频1”即可开始播放
2. **2.2文件夹为2.2相关代码**[^2]
    1. src为代码相关文件夹
        1. 根目录下为html
        2. js文件夹内为js
        3. css文件夹下为css
   1. 由于是插件且资源相关文件仅有一张图片所以图片也放在了src文件夹下，方便管理和使用。
3. **2.3文件夹为2.3相关代码**[^3]
   1. src为代码相关文件夹
        1. 根目录下为html及字体图标文件woff2
        2. js文件夹内为js
        3. css文件夹下为css
   2. res为资源相关文件夹
        1. photo文件夹下存放html中的图片
           1. photo下的cover文件夹为视频封面
        2. video文件夹下存放视频





[^1]:2.1为本地视频播放网页
[^2]:2.2为b站视频任意倍速chrome扩展
[^3]:2.3为搬到小破站
